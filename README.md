## Sobre

O Proffy é uma ferramenta onde é possível que professores façam a divulgação de suas aulas.

## Servidor

##### Instalando as dependências

```sh
yarn install
```

##### Instalando as migrations do banco de dados

```sh
yarn knex:migrate
```

##### Rodando local

```sh
yarn start
```
